# INCENDIES Profile Archive

Vite + React site.

## GitHub upload checklist

Upload these folders/files to the repository root:

- `index.html`
- `package.json`
- `vite.config.js`
- `src/`
- `public/`
- `README.md`

Do not upload an extra `.jsx` file to the repository root. The app entry is `src/main.jsx`, which imports `src/App.jsx`.

## Vercel

- Framework Preset: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
